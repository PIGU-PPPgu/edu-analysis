# 数据导入调试指南

## 快速排查

打开浏览器控制台（F12），在 Console 中运行：

```javascript
// 查看当前页面状态
console.log('当前路径:', window.location.pathname);

// 如果在导入页面，检查 localStorage
console.log('本地存储:', localStorage);
```

## 常见问题

### 1. "学号不能为空"错误
- 检查 Excel 第一列是否为学号
- 学号列是否有空值

### 2. "姓名不能为空"错误  
- 检查是否有姓名列
- 姓名列是否有空值

### 3. 字段映射问题
- 确保已完成字段映射步骤
- 每个系统字段都已映射到 Excel 列
