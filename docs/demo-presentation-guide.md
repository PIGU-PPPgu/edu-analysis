**Q1: 系统可以支持多少学生？**
A: 当前架构支持 10,000+ 学生，通过虚拟滚动和分页优化。如需支持更大规模，可以引入数据分片和缓存策略。

A: 使用 Supabase 的行级安全（RLS）+ HTTPS 加密传输 + 定期备份。符合教育数据保护规范。

**Q3: 可以私有化部署吗？**
A: 可以。支持 Docker 容器化部署，提供完整的部署文档和技术支持。

**Q4: 多久可以上线？**
A: 标准配置 2-4 周可上线，包括数据迁移、培训、试运行。定制需求需要评估。

**Q5: 费用如何计算？**
