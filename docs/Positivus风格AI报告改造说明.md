## 🎯 Positivus设计系统核心元素

### 1. **配色方案**
```css
Primary:   #B9FF66  /* 荧光绿/霓虹绿 - 视觉焦点 */
Secondary: #191A23  /* 深黑色 - 主要文字和边框 */
Accent:    #F7931E  /* 橙色 - 警示和点缀 */
White:     #FFFFFF  /* 背景和对比 */
```

**使用规则**：
- **荧光绿**：按钮主色、标题装饰、Logo背景、强调文字背景
- **深黑色**：所有边框、主要文字、表头背景
- **橙色**：警告信息、引用块、次要按钮阴影
- **白色**：卡片背景、内容区域

### 2. **标志性边框和阴影**

#### 粗黑边框
```css
border-2 border-[#191A23]  /* 标准边框 */
border-4 border-[#191A23]  /* 加粗边框（重要卡片）*/
border-8 border-[#191A23]  /* 超粗边框（封面元素）*/
```

#### 平移阴影（核心特征）
```css
/* 荧光绿阴影 - 主要元素 */
shadow-[4px_4px_0px_0px_#B9FF66]
shadow-[6px_6px_0px_0px_#B9FF66]
shadow-[8px_8px_0px_0px_#B9FF66]

/* 深黑阴影 - 按钮/标签 */
shadow-[4px_4px_0px_0px_#191A23]

/* 橙色阴影 - 警示元素 */
shadow-[4px_4px_0px_0px_#F7931E]
```

**悬停效果**：
```css
hover:translate-x-1 hover:translate-y-1
hover:shadow-[2px_2px_0px_0px_#B9FF66]
```
→ 阴影跟随元素移动，产生"按下"的感觉

### 3. **圆角系统**
```css
rounded-xl        /* 标准圆角 - 12px */
rounded-[20px]    /* 中等圆角 - 卡片 */
rounded-[30px]    /* 大圆角 - 大卡片/Logo */
rounded-full      /* 完全圆形 - 徽章/装饰 */
```

### 4. **字体粗细**
```css
font-medium   /* 400 - 普通文字 */
font-bold     /* 700 - 次要标题 */
font-black    /* 900 - 主要标题、按钮 */
uppercase     /* 大写 - 增强视觉冲击 */
```

## 🎨 Positivus风格要点总结

### ✅ **核心原则**
1. **高对比**：深黑 + 荧光绿 + 白色
2. **几何化**：直线、直角、圆形徽章
3. **粗线条**：2-8px的粗边框
4. **平移阴影**：4-8px的偏移阴影（无模糊）
5. **字体粗细**：大量使用font-bold和font-black
6. **全大写**：重要标题使用uppercase

### ✅ **视觉层次**
1. **最高层**：荧光绿徽章/按钮（shadow-[8px_8px...]）
2. **高层**：白色卡片（shadow-[6px_6px...]）
3. **中层**：内容区域（shadow-[4px_4px...]）
4. **基础层**：文字段落（无阴影）

### ✅ **交互反馈**
- **悬停**：translate + 阴影缩小
- **点击**：无需特殊效果（按下感已通过阴影实现）
- **过渡**：transition-all duration-200

### ❌ **避免使用**
- ❌ 渐变背景（除非特殊需求）
- ❌ 柔和阴影（drop-shadow）
- ❌ 过度圆角（45px以上除外）
- ❌ 低对比度配色
- ❌ 细字体（font-light, font-normal用得少）
